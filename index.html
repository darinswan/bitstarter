Moxy - Data Warehouse Database

There are several DW systems on the market today but many have drawbacks and limitations. They aren't really designed for optimal performance and don't cache data where they should resulting in more overhead than is needed to run the query. To make matters worse, the next query in the pipe does the same thing. This generates additional network bandwidth and unnecessary overhead. Moxy is a different approach that tries to run queries as fast as possible using various architectural approaches. It's not longer a one approach system for your data like many existing systems try to use.
